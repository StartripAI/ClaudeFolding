## 中文介绍

**Claude Code Folding** 是一个 clean-room 中文解剖项目。  
项目把公开可得的源码线索整理成可视化阅读体验，方便快速理解 Claude Code 的执行方式和系统边界。

站内内容包括：

- 11 步执行链
- 架构折叠图
- 工具权限层
- 命令簿
- 隐藏能力线索

你可以直接访问正式站点：

- [copilotinvoice.com](https://copilotinvoice.com)

## English

**Claude Code Folding** is a bilingual clean-room explainer for Claude Code.  
It turns publicly available source signals into a readable map of the execution loop, architecture, tools, commands, and hidden capability traces.

The site includes:

- the 11-step execution loop
- an architecture map
- tool and permission layers
- the command surface
- hidden feature signals

Live site:

- [copilotinvoice.com](https://copilotinvoice.com)
